# PHP Görev Yönetim Sistemi (MVC)

Bu proje, PHP dili kullanılarak Model-View-Controller (MVC) mimarisiyle geliştirilmiş bir görev yönetim uygulamasıdır.

## Özellikler
* Kullanıcı Kayıt ve Giriş Sistemi (Password Hashing)
* Görev Ekleme, Listeleme ve Silme
* MySQL Veritabanı Entegrasyonu
* Modern CSS Arayüzü
